1. Einleitung

" Zoom und Pan" um große Bilder (wie wir jetzt mit Maps umgehen): wenn wir unsere Perspektive nur ein wenig ändern, stellen Sie fest, dass eine Karte/Map ein Sonderfall eines riesigen Bildes ist, Sie haben eine Darstellung der Erde ein 3D-Objekt, das in eine 2D-Oberfläche projiziert wird. es ist sehr lage, dass man nicht das Ganze auf einmal betrachten kann, aber auch nicht den Detaillierungsgrad bekommen kann, an den man denken will, wenn man das Ganze betrachtet. So müssen Sie in der Lage sein, zu zoomen und um ihn herum zu schwenken, so dass Sie verstehen können, wo Sie sich auf der Sache befinden, und dass Sie die Teile betrachten können, die Sie jederzeit betrachten möchten, ohne den Überblick behalten oder in Ihrem Kopf behalten zu müssen, wie die ganze Sache ist. Das ist der Sinn von Google Maps, aber Sie können dies auch mit jedem Bild tun, das Sie wollen, mit bestimmten Tools, hier werde ich über einige dieser Zooming-Tools sprechen, die in Websites und mobilen Apps verwendet werden können.

2. Vorgeschlagene Technologien

OpenSeaDragon

Ein Open-Source, webbasierter Viewer für hochauflösende, zoombare Bilder, implementiert in reinem JavaScript, für Desktop und Mobile. Das bedeutet, dass Openseadragon der Teil des Dienstes ist, der es Ihnen ermöglicht, sich coole Zoom-Bilder anzusehen.

Supported Tile Sources

OpenSeadragon unterstützt mehrere Image-Serving-Protokolle wie :

  • Legacy Image Pyramids
  • IIIF (International Image Interoperability Framework)
  • Deep Zoom Images
  • OSM (Open Street Maps)
  • TMS (Tiled Map Service)
  • Zoomify
  • Custom Tile Sources
  • Simple Image

Features

OpenSeadragon bietet viele optionale Oberflächenelemente, die einfach über einfache Konfigurationseinstellungen aktiviert werden können. Darüber hinaus möchte OpenSeadragon es Ihnen ermöglichen, seine Steuerelemente an vollständig benutzerdefinierte Schnittstellen zu binden. wie z.B:

  • Zoom and Pan
  • Overlays
  • Viewport Navigator
  • Rotation
  • Toolbar
  • Binding Custom Buttons
  • Keyboard Navigation

Es kommt sehr häufig vor, dass Gruppen verwandter Bilder so dargestellt werden müssen, dass der Benutzer leicht durch sie navigieren kann. OpenSeadragon bietet hierfür einfache Optionen sowie vollständige Unterstützung für mehrere Bilder mit vollständiger Kontrolle

  • Sequence Mode
  • Image Reference Strip
  • Collection Mode
  • Full Multi-Image

Deep Zoom Composer

Deep Zoom ist eine Open-Source-Technologie, die von Microsoft bereitgestellt wird und Teil von Silverlight ist, mit der Benutzer ein großes Bild mit hoher Auflösung oder eine große Sammlung von Bildern schwenken und vergrößern können. Deep Zoom Composer Nehmen Sie Ihr Bild und geben Sie Ihnen die Bildpyramide. Es reduziert die für das erstmalige Laden erforderliche Zeit, indem nur der angezeigte Bereich und / oder nur die Auflösung heruntergeladen werden, in der er angezeigt wird. Nachfolgende Regionen werden heruntergeladen, während der Benutzer darauf schwenkt (oder hinein zoomt).

Um die Bilder zu sehen und zoomen und erkunden zu können, benötigen Sie OpenSeaDragon

OpenLayers

OpenLayers erleichtert das Einfügen einer dynamischen Karte in jede Webseite. Es kann Kartenkacheln, Vektordaten und Markierungen anzeigen, die aus einer beliebigen Quelle geladen wurden. OpenLayers wurde entwickelt, um die Nutzung von Geoinformationen aller Art zu fördern. Es ist völlig kostenlos, Open Source JavaScript. OpenLayer können nicht nur für Karten/Maps, sondern auch für statik Bilder verwendet werden.

Features

  • Tiled Layers
  • Vector Layers
  • Cutting Edge, Fast & Mobile Ready
  • Easy to Customize and Extend

EasyZoom

EasyZoom ist ein öffentlicher Dienst für Leute, die hochauflösende Bilder hosten und teilen möchten. Es unterstützt sowohl gelegentliche Dateiformate (jpg, tif, png, pdf, psd) als auch spezifische Mikroskopformate (svs, ndpi, scn, ...). Benutzer können problemlos große Dateien wie z. B. 2 GB mit einer Auflösung von 200.000 x 200.000 hochladen und diese dann in einem Browser anzeigen. Außerdem gibt es viele zusätzliche Funktionen wie das Hinzufügen von Anmerkungen, Kommentieren und Einbetten in externe Websites. Wir glauben, dass dieses Produkt vielen Fachleuten helfen kann, ihre riesigen Bilder im Internet zu teilen und mit anderen Menschen in ihren Interessengebieten zusammenzuarbeiten, sei es als Objektträger, Astronomiebilder oder einfach nur als großes Fotopanorama.

Features

  • Zooming: Hochauflösende Bilder herunterladen und einfach im Browser betrachten. EasyZoom unterstützt viele verschiedene Formate: JPG, NDPI, SCN, SVS, VMS, VMU, MRXS... Auch Designer können ihre Projekte in den folgenden Formaten herunterladen: PSD, PDF, PNG, PNG, TIFF.
  • Annotations: Gigapixel-Bilder Kommentieren, um ihnen eine neue Perspektive hinzuzufügen. es ist möglich Text, Bilder, Links oder Videos in Anmerkungen einzufügen.
  • Advanced sharing: Mit dem iFrame EasyZoom ist es möglich, Bilder auf jeder Website zu teilen und einzufügen.
  • Catalog : Catalogen erstellen nach Kategorien wie Kunst, Astronomie, Fotografie, Medizin, Mikroskopie, Grafikdesign...
  • Watermarking : Schutz des Originalinhalts durch Anbringen eines Wasserzeichens auf allen Ebenen Ihrer Bilder.
Easyzoom hat mehrere ausgezeichnete Funktionen, aber leider ist es nicht kostenfrei.
